
비동기란?
=> 동기적(순서대로 실행)이지 않은 것
=> 자바스크립트 싱글 스레드(코드를 처리할 수있는 일꾼이 딱 1먕)
=> A(1초) -> B(1초)-> C(3초) -> D(1초) // 6초
=> A -> B -> D
   C        //3초
=> 비동기가 탄생된 이유, 자바스크립트는 일꾼이 하나니까, 여러일을 동시에 해결하기 위해서
=> 동기적인 코드가 모두 실행되고 나서 실행햐애할 때

------

(비동기 처리가 된다)
